Standard di riferimento
-----------------------

Sono riportati di seguito gli standard tecnici indispensabili per l’applicazione 
delle presenti Linee Guida.

.. list-table:: Standard di riferimento
   :widths: 15 40
   :header-rows: 0

   * -    [X.509]
     -    Standard per la crittografia asimmetrica definito in RFC5280

   * -    [JWT]
     -    JSON Web Token definito in RFC7519

   * -    [JWT-BCP]
     -    JWT Best Current Practices definito in RFC8725

   * -    [JWK]
     -    JSON Web Key (JWK) in RFC7517
